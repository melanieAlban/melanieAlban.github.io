Quién está detrás del código

Soy desarrolladora Full Stack en formación, enfocada en construir aplicaciones completas — del modelo de datos y la API hasta la interfaz que usa la gente.

He trabajado tanto en el backend —APIs REST con Java y Spring Boot, contenerizadas con Docker y documentadas con Swagger— como en el frontend, desarrollando aplicaciones en Angular y Vue. Diseño bases de datos relacionales (MySQL, Oracle, SQL Server) y trabajo a diario con Git y GitHub para versionar y colaborar en equipo. Actualmente realizo una pasantía construyendo un sistema de reportería para una empresa del sector eléctrico.

Objetivo profesional

Especializarme en desarrollo de software, ciberseguridad y análisis de datos, aportando soluciones limpias, seguras y escalables.
